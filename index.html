📄 导入名单到公海

把探迹 / 阿里金品那种「一段一家公司」的自由文本名单一键导入公海，自动抽取 公司名 / 联系人 / 电话 / 地区 / 主营产品 / 阿里链接 / 外贸·股东联系人 等，按公司名去重，来源标记为文件名。多个名单可分次上传。

支持 .txt / .docx（新版 Word）/ .doc（老版 Word） 三种格式。

公海客户为独立资料库（已按公司去重），可与潜在客户同时存在同一公司。公共资料区 6 张表数据量较大，完整同步建议在 Notion 把每张表「导出 CSV」后用上方「导入公海 CSV」上传（已适配各表表头）。

📥 导入客户（Excel / CSV）

支持 Excel（.xlsx/.xls） 和 .csv 文件批量导入客户（CSV 自动识别 UTF-8 / GBK 编码）。表头需包含「公司名字」列（其余列名与系统字段一致即可自动匹配）。可先下载模板照着填。

📤 导出数据

导出为 CSV 可用 Excel 打开；导出 JSON 备份可完整保存全部数据（含客户与拜访关联），换电脑时用「恢复备份」导回。
